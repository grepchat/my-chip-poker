# CHIP — Калькулятор фишек для покера

Простое веб-приложение для расчета выплат по фишкам в покере.

## Возможности

- Ввод количества игроков
- Настройка стартового взноса и фишек
- Автоматический расчет выплат для каждого игрока
- Показ прибыли/убытка для каждого игрока
- Адаптивный дизайн для мобильных устройств

## Использование

1. Откройте [приложение на GitHub Pages](https://grepchat.github.io/my-chip-poker/)
2. Введите количество игроков
3. Укажите стартовый взнос и количество фишек
4. Введите финальные фишки для каждого игрока
5. Нажмите "Начать расчет"

## Технологии

- Чистый HTML, CSS и JavaScript
- Без зависимостей
- Работает офлайн после первой загрузки
